# 🌦️ Weather Report Dashboard (Power BI)

## 📊 Overview

This project is an interactive Power BI dashboard analyzing historical weather data (1984–2025). It provides insights into temperature trends, rainfall patterns, and climate behavior.

## 🎯 Key Features

* KPI Cards (Max, Min, Avg Temperature, Rainfall, Temp Range)
* Time-series analysis (Moving Average)
* Monthly & Yearly trends
* Rainfall vs Temperature analysis
* Insights & summary page

---

## 🛠️ Tools Used

* Power BI
* DAX
* Data Visualization

---

## 📈 Key Insights

* Temperature shows gradual increase over years
* Peak temperatures occur during mid-year
* Rainfall is seasonal with spikes
* Radiation influences temperature trends
